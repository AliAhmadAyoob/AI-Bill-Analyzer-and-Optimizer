# Smart Home Energy Optimizer — Karachi
FYP Project | Predicts electricity bill from appliance usage and recommends reductions.

---

## Setup — step by step

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

For OCR (reading bill images), also install Tesseract:
- Windows: https://github.com/UB-Mannheim/tesseract/wiki
- Ubuntu/WSL: `sudo apt install tesseract-ocr`

### 2. Run the notebooks (in order)
Open Jupyter:
```bash
jupyter notebook
```
- Run `notebooks/data_preparation.ipynb` → produces `daily_appliance_usage.csv`
- Run `notebooks/model_preparation.ipynb` → produces `backend/model/electricity_model.pkl`

### 3. Start the Flask backend
```bash
cd backend
python app.py
```
API will run at: http://localhost:5000

### 4. Open the frontend
Just open `frontend/index.html` in your browser.
(No separate server needed for the frontend.)

## API endpoints

| Method | Endpoint           | What it does                              |
|--------|--------------------|-------------------------------------------|
| POST   | /api/upload-bill   | OCR extract units from bill image/PDF     |
| POST   | /api/predict       | ML prediction from appliance inputs       |
| POST   | /api/optimize      | Generate reduction recommendations        |
| GET    | /api/appliances    | List of supported appliance types         |

---

## How it works (user flow)

1. User uploads their HESCO electricity bill (photo or PDF)
2. OCR reads units consumed and bill amount automatically
3. User enters their appliances and daily usage hours
4. ML model predicts energy consumption per appliance
5. Optimizer compares actual bill vs predicted and identifies biggest consumers
6. System recommends specific reductions with exact money savings
